Usage: localhost [options]

Options:
  -d, --directory <path>   Specify the directory to serve. (Ignored if Laravel is detected)
  -c, --cert <file>        Path to the SSL certificate file for HTTPS.
  -k, --key <file>         Path to the SSL key file for HTTPS.
  -p, --port <number>      Port to listen on. Defaults to 80 (HTTP) or 443 (HTTPS).
  -o, --open               Automatically open the server in the default web browser.
  -r, --redirect           Redirect all HTTP requests on port 80 to HTTPS (requires HTTPS).
  -i, --index <file>       Specify the index file to serve. (Ignored if Laravel is detected)
  -v, --verbose            Enable verbose logging (writes to php.log).
  -h, --help               Show this help message and exit.
